# ccpet

Claude Codeの動作状態をデスクトップ上のキャラクターアニメーションで表示するためのWindowsアプリです。
常に最前面に表示される透明ウィンドウで、ドラッグ移動可能。システムトレイから設定やアニメーションのカスタマイズもできます。

## ダウンロード

[Releases](https://github.com/musoukun/ccpet-release/releases) からインストーラーをダウンロードしてください。

- `ccpet_0.1.0_x64-setup.exe` — NSIS インストーラー
- `ccpet_0.1.0_x64_en-US.msi` — MSI インストーラー
